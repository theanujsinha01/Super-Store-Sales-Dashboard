

# Super Store Sales Dashboard

## Overview

This repository contains a Power BI dashboard (`Sales_pbi_dashboard.pbix`) that visualizes Super Store sales data. The dashboard provides high-level KPIs, category-wise insights, and year-over-year trends. It is designed to deliver quick business insights for decision-making.

## Dashboard Highlights

- **KPIs:** Total Sales, Quantity Sold, Profit, Avg Delivery Days  
- **Filters:** Region (Central, East, South, West)  
- **Sales by Sub-Category:** Phones, Chairs, Binders  
- **Sales by Category:** Technology, Furniture, Office Supplies  
- **Sales by Segment:** Consumer, Corporate, Home Office  
- **Sales by Payment Mode:** COD, Online, Cards  
- **Sales by Ship Mode:** Standard, Second Class, First Class, Same Day  
- **Monthly Performance YOY:**  
  - Monthly Profit (2019 vs 2020)  
  - Monthly Sales (2019 vs 2020)
